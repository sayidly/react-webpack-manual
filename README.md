# React APP from Scratch

Use node version with `v14.16.1`

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run dev
```

### Compiles and minifies for production

```
yarn run build
```

### Deploys the Project

```
yarn run deploy
```
